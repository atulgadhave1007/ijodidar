# docs/phases/

Sprint handoff documents live here, one per sprint, named `SPRINT_<N>_HANDOFF.md`
(e.g. `SPRINT_1_HANDOFF.md`). Use the template in `MASTER_PROMPT.md` §9.3.

For Pre-Sprint 0, name it `PRE_SPRINT_0_HANDOFF.md`.

This folder is empty until the first sprint closes — do not pre-create empty handoff files.
